## Stack objetivo

- Next.js
- React
- TypeScript
- TailwindCSS

## Reglas enterprise

- Consumir Auth, RBAC, UI, tipos y validaciones desde HD-CORE.
- No mezclar responsabilidades de otros dominios.
- No guardar datos sensibles en cliente.
- Auditar acciones sensibles.
- Revisar impacto cross-repo antes de cada cambio.
